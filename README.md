<h1>Mystique</h1>
<p>bot-clicker for adfly, shrinkme, etc...</p>
<h3>by ph.phoenix</h3>
<hr>

<h2>requirements:</h2>
<ul>
<li>proxychains</li>
<li>Tor</li>
<li>firefox</li>
<li>python 3</li>
</ul>

<h2>Installation</h2>
<h4>On termux</h4>
<ul>
<li>pkg install tor</li>
<li>pkg install proxychains</li>
<li>pkg install python3</li>
<li>git clone https://www.github.com/ph-phoenix/mystique.git</li>
</ul>
<h4>On linux</h4>
<p>(nah, you already knew it.)</p>

<h2>usage:</h2>
<ul>
<li>Open Firefox first</li>
<li>service tor start</li>
<li>cd mystique</li>
<li>chmod +x bot.py</li>
<li>python3 bot.py</li>
</ul>
